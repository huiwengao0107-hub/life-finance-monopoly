人生理財大富翁 V5.0 隊伍設定版

本版依照原 V3.3 資料升級，未更動既有棋盤路徑、卡牌內容與核心規則。

新增：
1. 遊戲開始前設定三組自訂隊名。
2. 每組可選擇代表圖示與隊伍顏色。
3. 棋子、目前回合、資產區、主持人選單與排行榜同步顯示自訂隊名。
4. 隊名不可空白或重複。
5. 保留每組起始現金 $5,000，新手任務完成後加 $2,000。
6. 保留理財知識卡先顯示題目、主持人再揭曉答案。
7. 保留合作挑戰連動情境卡與理財知識卡。
8. 保留 START→1~25→重複1~13→GOAL 的固定路徑。

使用方式：
以 Chrome 或 Edge 開啟 index.html。
本版為單機主持人可測試版本；真正跨手機即時同步仍需 Firebase 專案與網路部署。
